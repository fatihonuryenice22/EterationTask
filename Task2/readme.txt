this postman_dataset.json is prepared for runner in postman collection with this dataset you can run your POST REQS.
Steps
1- Go to postman collection link
2-Click to POST REQS folder 
3- On the right top of the page Run button will be visible, click to it
4-You will see data file option select your data file as a postman_dataset.json
5- run the collection
6- Results will appear in a short time






Another note: Negative Tests are carried to Bonus Task.
